# Fold Map Template (diegetic UI)

Fold's map is the only place where stats, keys, and rules are displayed.

Zones:
1) Route Keys row (8 slots top)
2) Biome stamp area (left margin)
3) Rule Tag pin (upper-right)
4) Stats stamp strip (bottom)
5) Continuity notes: fresh/frayed/cracked/mended (no numbers)
